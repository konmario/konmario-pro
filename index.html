Worum es geht

Konmario-pro ist eine private Mac-App: ein Fenster, 40 Werkzeuge — vom Download-Sortierer über ein ADHS-freundliches Haushaltsbuch bis zum Fehler-Doktor, der Screenshots von Fehlermeldungen per KI erklärt. Dazu eine Menüleisten-Version, ein schwebendes Desktop-Cockpit und ein Nachtschicht-Helfer, der stündlich von selbst aufräumt.

Das Besondere: Fast alles läuft lokal. Die Stimmen (17 Stück, von Apples Klassikern bis zur KI-Stimme Kerstin, die komplett auf dem MacBook wohnt), die Spracherkennung (Whisper tippt Diktate ab, offline), die Texterkennung (Apples eingebaute Vision-KI). Kein Abo, keine Cloud-Pflicht, keine laufenden Kosten — auf Hardware, die offiziell längst ausgemustert ist. Beliefert wird, wer sich beliefern lässt.

Die Chronik — eine Woche im Zeitraffer

Tag 1 · Die Geburt

Aus zwei Skripten in einem unordentlichen Ordner wird eine App. Das eingebaute Fenster-Modul stürzt sofort ab — aus der Not entsteht die Architektur: lokaler Server, Browser als Anzeige. Wie bei den Großen.

Tag 2 · Die Stimme

Die Toolbox lernt sprechen — erst mit Apple-Stimmen, dann mit einer lokalen KI-Stimme (Piper), am Ende mit 17 Stimmen aus vier Technologien. Dazu: Menüleisten-Icon und die erste Nachtschicht.

Tag 3 · Das System

Das ADHS-Set entsteht: Fokus-Timer mit Stimm-Ansage, Entscheidungs-Würfel, Gehirn-Parkplatz, Haushaltsbuch und Mini-Tagebuch mit echten Datenbanken. Ein Feld, zwei Sekunden, fertig.

Tag 4 · Die Sinne

Umzug auf modernes Python — und die Toolbox lernt hören (Whisper, lokal) und sehen (Texterkennung, Fehler-Doktor). Der Werkzeug-Schmied übernimmt die Zukunft: Werkzeuge bauen jetzt Werkzeuge.

Tag 5 · Das Schaufenster

konmario-pro.de geht live — GitHub Pages, eigene Domain, DNS von Hand verkabelt. Diese Seite, die du gerade liest, wird von einem Toolbox-Werkzeug generiert. Der Kreis schließt sich.
